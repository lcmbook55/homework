#### 03. CHOICE

![image](https://postfiles.pstatic.net/MjAxOTExMTFfMjY1/MDAxNTczNDYxMjI4NjAx.SKNnTgvdSbmeK-4irgioQH0Gk5RIPl1uYS7xC2Jmy-sg.OD92fwiRRU1CKwWseHNW4zIm-cpQi9mqlA8_vXwvjZ0g.JPEG.rearrange12/%EC%98%A4%EB%9D%BC%ED%81%B4_3.JPG?type=w773)

Github 수업을 무사히 마치고 나는 작은 선택을 해야했다.<br>

`go?` `stop?`<br>

약간 엄살을 부리며 강사쌤에게 난이도를 물어봤다. 중간 난이도.
내가 듣고싶은 말이였다. 만약 난이도 '하'였다면 아마 깊은 생각에 빠졌을거다.<br>

갑자기 인생을 논해서 좀 웃긴데, 인생은 모든것이 선택이다.<br>
선택을 잘 하기위해, 후회를 하지 않기위해, 더 지혜로워지기 위해 사람들은 많은 고민을한다.<br>

이번 homework할때도 수많은 선택들이 있었다.<br>
마크 문법 숙제를 마치고 git push 까지한 상태에 결과물을 봤을때 띄어쓰기, 오타 등등 수정할 것들이 좀 있었다.
preview를 함께 보면서 작업을 한다고 해도 수정해야할 부분들을 다시 한번 확인해야된다는 것을 알았다.<br>

이 회고록의 컨셉도 한 두번 갈아엎으면서 다양한 선택들을 할수 있다는 것이, 바로 내 눈앞에 보여지는 결과물에 대해서 흥미가 생겼다. 더 나아가 강사님의 버튼을 눌러서 문법 예시를 보여준다거나, 이미지 사이즈를 변경한다거나 하는 여러가지 아이디어가 생각났다. 재밌는데?<br>

이미 선택을 했고, 앞으로 계속 선택을 해야할것이다.

#### 04. PRESS START

![image](https://i.pinimg.com/1200x/78/49/2a/78492a7b117bcbdacecbb5a5a8e8587d.jpg)

결론은 마크다운 문법으로 정리를 하면서 재밌는 선택들이 모이고, 한번 정리하고나니 자신감이 생겼다. 세번의 오타수정과 git commint으로 push하는 과정도 꽤 손에 익고있다.<br>

강사쌤과 함께 코드를 입력할때는 너무 잘 되다가 혼자 다시해보면 계속 실패가 뜬다.<br>
수업내내 어리둥절 너무 정신이 없지만 수업이 끝난뒤에는 `있다가 없습니다?` `없었다가 있습니다?` 처럼 수업 내용들이 머릿속에 굴러다닌다. 참 이상하다.

수업시간에 많은 질문들이 있지만, 강사쌤께서 바로 피드백을 주셔서 너무 감사하다.<br>
최대한 나는 질문하기 전에 내가 먼저 해내고 싶어서 AI에게 물어보거나, 수업자료를 통한 스텝바이스텝으로 진행하고있다. 아직까지는 혼자의 힘으로 버틸만하다.<br>
내가 해결할 수 있는 것은 최대한 내 힘으로 해결해보고싶다. 조금 느리더라도!<br>
